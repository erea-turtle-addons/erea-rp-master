# E'rea RP Master

A Game Master tool for running E'rea RP events on Turtle WoW.

## What it does
- Create and manage custom RP items with descriptions, icons, and actions
- Give items to players, trigger cinematics, and run scripted event sequences
- Monitor nearby players and their RP inventories during events

## Requirements
- GM or event organizer use only — not needed by regular players
- Requires **E'rea RP Player** to be installed alongside it

## Commands
- `/rpmaster` — open the RP Master panel
